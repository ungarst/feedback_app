It looks like you have tried to call a method, but you have included the types of the inputs in the method call.

It is unnecessary to do this, as Java already knows what type the inputs are from the definition of the method you are calling. Remove the variable types from in front of the inputs in the method call and try again.

Incorrect Code Correct Code Explanation

Example

 
String word = "Programming";
int position = 4;
char end = word.charAt(int position);
 
String word = "Programming";
int position = 4;
char end = word.charAt(position);

We don't need to specify that position is an int in the call to charAt().

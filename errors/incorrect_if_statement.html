It appears that you have an incorrectly written if statement in your code.

Lets quickly review the format of an an if statement and see if it can help you fix your error.

if (condition) { 
	// Statements
} else if (condition) { 
	// Statements
} else { 
	// Statements
}

The condition and corresponding block of statements are the only required parts.

It is also possible to have:

  • if statements
  • if, else statements
  • if, else if, else statements
  • if, else if statements

Ensure that all your if statements follow this format and then try compile your code again

Incorrect Code Correct Code Explanation

Example

 
int c = 4;
if (c > 6); { 
	return true;
} else { 
	return false;
}
 
int c = 4;
if (c > 6) {
	return true;
} else { 
	return false;
}

The semicolon after the first condition actually finishes the if statement. It is a bit like saying: “if c is greater than 6, then do nothing”. All of the following code is unrelated to the if statement, so the compiler gets confused when it sees the “else” keyword.

You are probably missing an operator in your code:

A lot of the time, this could just be a missing assignment operator (i.e. the equals sign), but it can also be other operators like +, -, /, *, %, || and &&.

If you are trying to assign a value to a variable, then you should use the assignment operator (i.e. put an “=” sign after the variable name and before the value).

Otherwise, use the appropriate arithmetic operator (+, -, /, *, %) if you are calculating something, the appropriate boolean operator (||, &&) if you are combining logical expressions, or the comparison operator (==) is you are comparing two values.

It appears you are using the increment (++) or decrement (--) operators incorrectly:

These operators provide a simple way to increase or decrease the value of a variable by one. The variable must be of a numeric type, and it must be initialised.

Example

i++;

String s = "Programming";

int i = 0;

Variables need to be initialised before they can incremented or decremented, so i++ is not correct. Strings cannot be incremented, and so s++ is not allowed.

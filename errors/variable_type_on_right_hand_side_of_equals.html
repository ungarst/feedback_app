It appears that when initializing one of your variables you have included the variable type on the right hand side of the equals sign.

You don't need to do this as Java can automatically determine the correct type of the expression on the right hand side. Remove the variable type from the right hand side of the equals sign and try again.

Incorrect Code Correct Code Explanation

Example

 
int a = int 7; 
int a = 7;

We don't need to include the type (i.e. int) in front of the 7

It looks like you are trying to use a variable somewhere in your code without first having declared the variable.

Remember, every variable must first be declared (i.e. where you specify the type of the variable and give it a name) before you try to use it or assign it a value. This is so that Java can check you are using the variable in the correct way based on its type.

Incorrect Code Correct Code Explanation

Example

 
a = 5;
 
int a = 5;

a has a value assigned to it, but has never specified what type of variable it is

Also, remember to be careful of the scope of your variables.

If your variable is declared within a block of code (e.g. inside the body of an if statement), then it will only be accessible within that block.

If you want to access your variable outside of that block, then you will need to declare it before the block.

Incorrect Code Correct Code Explanation

Example

 
int x = 10;
if (x > 5) {
	int y;
	y = x + 4;
}
System.out.println(y); 
int x = 10;
int y;
if (x > 5) {
	y = x + 4;
}
System.out.println(y);

In the code on the left, the variable y cannot be printed in the last statement as it is only accessible within the body of the if statement. The scope can be changed, and this problem corrected, but declaring y before the if statement.

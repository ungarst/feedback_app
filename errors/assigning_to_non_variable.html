It appears that you are attempting to assign to something that is not a variable.

Make sure that on the left hand side of an equals sign (the assignment operator) you have a variable. The value of the expression on the right hand side will then be assigned, or stored, in this variable.

Incorrect Code Correct Code Explanation

Example

 
int a, b;
a + b = 10;
 
int a, b;
b = 100;
a = b + 10;

The equals sign is the assignment operator, and so the left hand side of the equals sign must be a variable, not an expression.

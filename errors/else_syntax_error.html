It appears that one of your if statements is written incorrectly.

Lets quickly revise the correct structure of an if statement

if (condition) { 
	// Statements
} else if (condition) { 
	// Statements
} else { 
	// Statements
}

The most likely cause of your error is having a semicolon after the condition (see the example for a full explanation).

Ensure that all your if statements are written correctly and then try run your code again.

Incorrect Code Correct Code Explanation

Example

 
int c = 4;
if (c > 6); { 
	return true;
} else { 
	return false;
}
 
int c = 4;
if (c > 6) {
	return true;
} else { 
	return false;
}

The semicolon after the first condition actually finishes the if statement. It is a bit like saying: "if c is greater than 6, then do nothing". All of the following code is unrelated to the if statement, so the compiler gets confused when it sees the "else" keyword.

It appears your code encountered a type mismatch error.

Type mismatch errors are caused by trying to assign a value to a variable, where the type of the value does not match the type of the variable.

Make sure that whenever you make an assignment (using the = operator), the type of the expression on the right hand side matches the type of the variable on the left hand side of the operator.

Incorrect Code Correct Code Explanation

Example

 
boolean b = false;
int x = 1;
b = x;
 
boolean b = false;
int x = 1;
b = (x != 0);

Boolean variables can only store true or false. In the code on the left, we attempt to store a value of type int (the value 1) in a variable of type boolean. This type mismatch is not allowed.

It appears that you have an incorrectly formatted for loop in your code.

The only requirements of a for loop is having two semicolons in parenthesis, but the correct structure of a for loop is:

	for(initialization; condition ; update)
	{
	   // statements
	}

Where:

  • The initialization declares a loop variable and sets it to an initial value.
  • The condition is a boolean expression and the body of the for loop will be executed only when this expression evaluates to true
  • The update changes the value of the loop variable each time through the loop

Note that two semi-colons must be used to separate each of these three sections.

It is possible to rearrange this so the initialization and upadate are not within the parenthesis but this is the recommended way.

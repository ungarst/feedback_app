It appears that you are trying to assign to a variable incorrectly, and are somehow missing, or incorrectly using an equals sign.

Remember, variable assignment should look like: variableName = value; (if the variable has already been declared) or variableType variableName = value; (this declares and assigns to the variable)

Incorrect Code Correct Code Explanation

Example

 
int x = 5;	

int a 10; 

boolean x == 10;
 
int x = 5;

int a = 10;

boolean b = (x == 10);

On the second line we need to insert an equals sign so we are assigning 10 to a. On the third line we need to give the boolean a name so we are assigning to something.

Programmers often make this mistake because they are trying to access an element in an array, but forget to put the index of the element they are trying to access in the square brackets.

int second = a[]; 
int[] a = {1, 2, 3, 4, 5};
int second = a[1];

We want to access the second element of the array a, but we forget to put the index in the square braces

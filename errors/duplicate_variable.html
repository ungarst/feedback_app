It seems like you are trying to create two variables with the same name in your method.

Remember, once you have declared a variable (i.e. defined its type and name) you can assign to it in multiple places.

Incorrect Code Correct Code Explanation

Example

 
int a = 5;
...
int a = 10;
 
int a = 5;
...
a = 10;

The first line declares the variable a (and initialises its value to 5). We should not declare the variable again if we want to assign a new value to it.

This error will also occur if you try to declare any variables with the same name as the inputs to the method you are writing. The input variables have already been declared and store the values given to them when the method was called.

Example

 
public static int sum (int a, int b) {
	int a, b;
	return a + b;
}
 
public static int sum (int a, int b) {
	return a + b; 
}

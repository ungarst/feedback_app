It appears you have broken the Java naming conventions.

The full Java naming conventions can be found in the Java Tutorials but we've included a brief summary of them below which should help you solve your problem.

  • Variable names are case-sensitive.
  • Variable names can contain letters, numbers and underscores.
  • Variable names must begin with a letter.
  • Variable names must not be one of the Java keywords (e.g. int, new)

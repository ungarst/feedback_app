Looking at your code and the error that Java found, it appears that you are are performing a comparison using a single equals sign.

Remember, if you want to test if two things are equal you need to use two equals signs.

One equals sign assigns the value on the right to the variable on the left.

Incorrect Code Correct Code Explanation

Example

 
int a = 10;
int b = 8;
if (a = b) { 
	return true;
} else {
	return false;
}
 
int a = 10;
int b = 8;
if (a == b) { 
	return true;
} else {
	return false;
}

The statement within the if statements brackets assigns the value of b to a. If we want to compare them we need to use two equals signs (the comparison operator).

It appears that you are trying to create a variable of a type that does not exist.

This error is usually caused by incorrectly spelling the type of variable you are declaring.

Incorrect Code Correct Code Explanation

Example

 
Int x = 10;
string s = "hello";
 
int x= 10;
String s = "hello";

ints need to be declared with a lower case 'i' and strings need to be declared with an uppercase 'S'.

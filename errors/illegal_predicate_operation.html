It looks like you are using an illegal operator in one of your boolean expressions.

Operators in boolean expressions that are often used incorrectly include:

  • =! - The exclamation mark should be before the equals sign
  • => - The greater than sign should come before the equals sign
  • =< - The less than sign should come before the equals sign
  • << - This is the left bitwise shift, not much less than
  • >> - This is the right bitwise shift, not much greater than
  • \\ - Remember, if you are trying to have your if based on one condition OR an other, you need to use ||
Incorrect Code Correct Code Explanation

Example

 
int x = 5;
if (x =< 10) { 
	x = 10;
}
 
int x = 5;
if (x <= 10) { 
	x = 10;
}

The less than and equals signs are around the wrong way. By correcting their order the code can now compile.

It looks like you have forgotten to wrap the boolean condition for an if statement in parentheses.

Just place an opening and closing brace around the boolean condition: if the condition evaluates to true, then the body of the if statement will be executed.

Incorrect Code Correct Code Explanation

Explanation

 
int a = 10;
if a == 5 {
	return true;
}
 
int a = 10;
if (a == 5) {
	return true;
}

The boolean condition for the if statement (a==5) needs to be wrapped in brackets.

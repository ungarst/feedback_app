It appears that there is an error with the way you are assigning values.

Remember, assignment to a variable follows the form

  • variableType variableName = value; if you haven't already declared the variable.
  • variableName = value; if you have already declared the variable type.
Incorrect Code Correct Code Explanation

Example

 
int hours = 0;
hours + 10;
 
int hours = 0;
hours = hours + 10;

In the code on the left, the statement "hours + 10" is incorrect as there is no assignment operator (i.e. equals sign). This expression would be evaluated, but the result is not stored anywhere. Changing this to "hours = hours + 10" means the value of the expression is stored in the variable "hours", replacing the old value.

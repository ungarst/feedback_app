It appears you attempted to return multiple things packaged in parenthesis in your code.

Java only allows us to return one thing. This must also have the same type as the return type specified in the method header.

Incorrect Code Correct Code Explanation

Example

 
int a = 12;
String s = "Java";
return (a, s);
 
int a = 12;
String s = "Java";
return a;

Java does not let us return multiple things. Instead we must return a or return s. We can only return one thing that is the same type as the return type specified in the method header

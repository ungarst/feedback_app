It appears one of your if statements isn't written properly.

When we try to compare two things in an if statement, we need to use two equals signs.

Remember:

  • One equals sign is for assigning a value to a variable e.g. int x = 10
  • Two equals signs are for comparing two things e.g. x == 10
Incorrect Code Correct Code Explanation

Example

 
int x = 5;
int y = 13;
if ((x*y)%2 = 0) {
	return true;
} else {
	return false;
}
 
int x = 5;
int y = 13;
if ((x*y)%2 == 0) {
	return true;
} else {
	return false;
}

In the code on the left, a single equals sign appears in the condition for the if statement. This is incorrect, because a single equals sign is the assignment operator. To compare two values we must use two equals signs (==) which is the comparison operator, as shown on the left.

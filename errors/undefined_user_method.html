It appears you are trying to call a method that doesn't exist.

Let's see if we can fix this... there are two different kinds of methods that you can call from the code you are writing for this exercise:

  • static methods from a library class (e.g. Math.max(), Integer.parseInt())
  • instance methods that are called on variables you have defined in the method you are writing (e.g. variableName.methodName(parameters))
Incorrect Code Correct Code Explanation

Example

 
int a = 5; 
int b = 12;
int c = max(a,b);
 
int a = 5; 
int b = 12;
int c = Math.max(a,b);

max is in the Math package so we need prefixed it with the package name

Also note that this error can be caused by misspelling Java keywords that are followed by parentheses. For example, if you are trying to write an "if" statement but spell it "is" (e.g. is (condition)) Java will think this looks like a method call, but the "is" method does not exist!

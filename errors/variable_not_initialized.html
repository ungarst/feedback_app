It appears one of your variables is being accessed without being initialized.

This is generally a result of declaring a variable without giving it an initial value and then trying to access its value later on.

Common occurences of this error are:

  • Using a for-loop to generate the sum of an array, but not setting the sum variable to zero before the for-loop (see below).
  • Using a while-loop, but forgetting to set the iterating variable.
Incorrect Code Correct Code Explanation

Example

 
int[] a = {4, 64, 92, 13, 9};
int sum;
for (int i = 0; i < a.length; i++) {
	sum += a[i]; 
}
 
int[] a = {4, 64, 92, 13, 9};
int sum = 0; 
for (int i = 0; i < a.length; i++) {
	sum += a[i];
}

We cannot add a[i] to sum as we never specified what sum was. By initializing sum to be 0 before the for loop we can add to it in the for loop.

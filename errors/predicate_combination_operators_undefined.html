It appears you are incorrectly using one or more of the following operators in your code:

  • &&
  • ||
  • !

Both the && operator and the || operator can only appear between two boolean values. The ! operator can only appear to the left of a boolean value.

Most occurrences of this error are caused by incorrectly formed boolean expressions (for example, in the condition of an if statement).

Incorrect Code Correct Code Explanation

Example

 
double items = 100.0/9.0;

if (items <= 10 && !0) { 
	return false;
}
 
double items = 100.0/9.0;

if (items <= 10 && items != 0) { 
	return false;
}

The code on the left is incorrect because the ! operator can only appear to the left of an expression of boolean type (but 0 is of type int). In addition, the && operator can only appear between two expressions of type Boolean.

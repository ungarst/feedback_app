It appears your code is generating an OnlySynchronized error. This is generally due to a syntax error in an if statement.

Remember that the condition for an if statement must be surrounded by opening and closing parentheses.

if (condition)

This is true even if the condition consists of more than one boolean expression combined with logical operators like && or ||.

Incorrect Code Correct Code Explanation

Example

 
double items = 9.4;
int a = 6;

if (items >10) && (a == 0) {
	return true;
}
 
double items = 9.4;
int a = 6;

if ((items >10) && (a == 0)) {
	return true;
}

The condition in an if statement needs to be enclosed in parentheses. Even if this condition is made up of the combination of other conditions the entire thing still needs to be wrapped in parentheses

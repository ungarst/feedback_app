It appears you are returning a value that is a different type to the type specified in the method header.

Look at the return type in the method header and make sure that what you are returning is the same type.

Incorrect Code Correct Code Explanation

Example

 
public boolean isEven (int a)
{
  return a;	
}
 
public boolean isEven (int a)
{
  boolean b = (a%2)==0;
  return b;
}

The method header says that we need to return a boolean, but a is an integer.

It appears as though you have a variable declaration, but you have wrapped the type of the variable in parentheses.

Remove the parentheses from around the variable type and try again.

Incorrect Code Correct Code Explanation

Example

 
(int) i = 50;
(String) s = "Hello";
 
int i = 50;
String s = "Hello";

The types in variable declarations do not require parenthesis around them.

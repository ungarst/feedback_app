It appears that you have an out of place semicolon in your code. This is generally caused by one of two things:

  • Having an extra semicolon in your code. This most commonly occurs in at the end of a line where the programmer puts a semicolon before the last closing parenthesis (see example below).
  • Incorrect variable declaration where the programmer forgets to specify the variable type in front of the variable name.
Incorrect Code Correct Code Explanation

Example

 
int product = 8*7;
if ( product % 2 == 0;) {
	return true;
}
return false;
 
int product = 8*7;
if ( product % 2 == 0) { 
	return true;
} 
return false;

Semicolon at the end of the comparison is not required

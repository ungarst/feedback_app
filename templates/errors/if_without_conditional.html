It appears one of your if statements is missing a condition which is stopping your code from compiling.

If statements require a condition so they will only be executed under certain circumstances.

To fix this, ensure that all of your if statements have a boolean condition in brackets after the keyword if and before the opening curly brace.

Incorrect Code Correct Code Explanation

Example

 
if {
	// Statements
}
 
if (condition) {
	// Statements
}

It looks like your code was unable to compile. Lets take a look why...

Looking at your code and the error that Java found, it appears that you are are performing a comparison using a single equals sign.

Remember, if you want to test if two things are equal you need to use two equals signs.

One equals sign assigns the value on the right to the variable on the left.

Example

Incorrect Code

int a = 10;
int b = 8;
if (a = b) { // This line assigns the value of b to a, and resolves to an int not a boolean. If we want to compare them we need to use two equals signs.
	return true;
} else {
	return false;
}

Correct Code

int a = 10;
int b = 8;
if (a == b) { // This line assigns the value of b to a, and resolves to an int not a boolean. If we want to compare them we need to use two equals signs.
	return true;
} else {
	return false;
}

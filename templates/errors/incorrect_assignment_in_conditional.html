It appears that you are incorrectly assigning to a variable in one of your if statements.

Assignment in if statements must happen before the value of the variable is used in a comparison. Most of these kind of errors occure when people forget to put the assignment first or put their brackets in the wrong place (see example below).

Ensure that you have your assignment and comparisons in the right order and your brackets are correct, then try run your code again.

Incorrect Code Correct Code Explanation

Example

 
String word = "10";
int x;
if ((x = Integer.parseInt)(word < 3)) {
	return true;
} else {
	return false;
}
 
String word = "10";
int x;
if ((x = Integer.parseInt(word)) < 3) {
	return true;
} else {
	return false;
}

Integer.parseInt(String s) takes in a word and converts it to an integer. We need to re arrange the braces so we are correctly assigning to a variable, and then we can compare the value of the variable.

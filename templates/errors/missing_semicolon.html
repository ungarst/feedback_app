It looks like your code was unable to compile. Lets take a look why...

It appears that that one of your statements is missing a semicolon at the end.

Remember, every statement needs to be completed with a semicolon.

Examples

Incorrect code

int a = 5
System.out.println("Hello, World")

Correct code

int a = 5;
System.out.println("Hello, World");

However, lines that start a block of code do not require a semicolon after them.

Instead, they should be completed with and opening brace.

Example

Incorrect code

if (x > 5) 
	return true;
}

Correct code

if (x > 5) {
	return true;
}

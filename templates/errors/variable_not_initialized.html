It looks like your code was unable to compile. Lets take a look why...

It appears one of your variables is being accessed without being initialized.

This is generally a result of declaring a variable without giving it an initial value and then trying to read it later on.

Common occurences of this error are:

  • Using a for-loop to generate the sum of an array, but not setting the sum variable to zero before the for-loop (see below).
  • Using a while-loop, but forgetting to set the iterating variable.

Example

Incorrect Code

int[] a = {4, 64, 92, 13, 9};
int sum;
for (int i = 0 ; i < a.length ; i++) {
	sum += a[i]; // Trying to add a[i] to the current value of sum, but sum has no current value
}

Correct Code

int[] a = {4, 64, 92, 13, 9};
int sum = 0; // Sum starts as 0 so we can add a value to the current value of sum
for (int i = 0 ; i < a.length ; i++) {
	sum += a[i];
}

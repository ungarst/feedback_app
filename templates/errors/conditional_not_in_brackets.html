It looks like your code was unable to compile. Lets take a look why...

It looks like you are forgetting to wrap your if condition in parenthesis.

Java requires you do to this so it knows what to evaluate as the if condition, determining if it gets executed or not.

Make sure that you have parenthesis around the conditions in your if statements and then try run your code again.

Example

Incorrect Code

int a = 10;
if a == 5 {
	return true;
}

Correct Code

int a = 10;
if (a == 5) {
	return true;
}

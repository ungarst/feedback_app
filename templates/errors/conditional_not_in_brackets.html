It looks like you are forgetting to wrap your if condition in parenthesis.

Java requires you do to this so it knows what to evaluate as the if condition, determining if the following code gets executed or not.

Make sure that you have parenthesis around the conditions in your if statements and then try run your code again.

Incorrect Code Correct Code Explanation

Explanation

 
int a = 10;
if a == 5 {
	return true;
}
 
int a = 10;
if (a == 5) {
	return true;
}

The boolean condition for the if statement (a==5) needs to be wrapped in brackets.

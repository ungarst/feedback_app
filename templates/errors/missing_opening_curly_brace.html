It appears your code is missing a '{'

Remember, all blocks of code such as methods, loops and if statements need to be opened with a '{' and closed with a '}'

Incorrect Code Correct Code Example

Example

 
if (x > 5) // No '{' to open block of code
	x = x + 5;
	sum += x;
}
 
if (x > 5) { // '{' successfully opens the block of code
	x = x + 5;
	sum += x;
}

The if statement has no '{' to open the block. Inserting a opening curly brace successfully opens the block of code.

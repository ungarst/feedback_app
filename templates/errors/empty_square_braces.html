It looks like your code was unable to compile. Lets take a look why...

It appears that you have empty square brackets in your code, which is stopping your code from compiling properly.

Programmers generally make this mistake because they:

  • Are trying to access an element in an array, but forget to put the index of the element they are trying to access in the square brackets.
  • Are trying to convert a string to an array of characters, but don't know how to do this properly.

Example of forgetting index

Incorrect Code

// We want to access the second element of the array a
int[] a = {1, 2, 3, 4, 5};
int second = a[]; // Forget to put the index in the square braces

Correct Code

// We want to access the second element of the array a
int[] a = {1, 2, 3, 4, 5};
int second = a[1]; // Forget to put the index in the square braces

Example of converting a string to an array of characters

Incorrect Code

String s = "I love programming!";
char[] characters = s[]; // Using this will not create an array of characters

Correct Code

String s = "I love programming!";
char[] characters = s.toCharArray(); // Correct use of .toCharArray() to create array of characters

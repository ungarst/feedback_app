It appears that you have empty square brackets in your code, which is preventing your code from compiling properly.

Programmers generally make this mistake because they:

  • Are trying to access an element in an array, but forget to put the index of the element they are trying to access in the square brackets.
  • Are trying to convert a string to an array of characters, but don't know how to do this properly.
Incorrect Code Correct Code Explanation 
int[] a = {1, 2, 3, 4, 5};
int second = a[]; 
int[] a = {1, 2, 3, 4, 5};
int second = a[1];

We want to access the second element of the array a, but we forget to put the index in the square braces

Example

 
String s = "I love programming!";
char[] characters = s[]; 
String s = "I love programming!";
char[] characters = s.toCharArray();

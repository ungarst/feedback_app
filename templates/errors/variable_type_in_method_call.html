Looking at your code, it appears that you are calling a method including the variable type with the variable type in the parameters (e.g. someMethod(int 5); ).

It is unnessasary to do this, as Java already knows what type the variables are from when they were declared originally. Remove the variable types from in front of the variable names in the method calls and try run your code again.

Incorrect Code Correct Code Explanation

Example

 
String word = "Programming";
int position = 4;
String end = word.substring(int position);
 
String word = "Programming";
int position = 4;
String end = word.substring(position);

We don't need to specify that position is an int in the call to substring.

It appears that one of your if statements is written incorrectly. Lets quickly revise the correct structure of an if statement

if (condition) { 
		// Statements
	} else if (condition) { 
		// Statements
	} else { 
		// Statements
	}

The most likely cause of you error is having a semicolon after the if condition (see the example for a full explanation).

Ensure that all your if statements are written correctly and then try run your code again.

Incorrect Code Correct Code Explanation

Example

 
int c = 4;
if (c > 6); { 
	return true;
} else { 
	return false;
}
 
int c = 4;
if (c > 6) {
	return true;
} else { 
	return false;
}

The semicolon after the if statement finishes the conditional area, and so all code after it will be executed regardless. The following opening curly brace opens a new block of code, which will always be executed. When it reaches the matching closing curly brace, the block of code is completed and execution will continue. The program then reaches the else condition, but it isn't preceeded by an if statement. As all else statementsneed to be preceeded by an if statement, Java throws an error.

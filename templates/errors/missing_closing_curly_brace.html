It appears you are missing a '}' in your code.

Remember, all blocks of code such as methods, loops and if statements need to be opened with a '{' and closed with a '}'

Incorrect Code Correct Code Example

Example

 
if (x > 5) {
	x = x + 5;

if (x > 5) {
	x = x + 5;
}

The last line has no '}' to finish the block. Inserting a closing curly brace successfully closes the block of code.

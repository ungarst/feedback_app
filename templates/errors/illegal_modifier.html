It appears that you are using an illegal modifier on one of your variables, most likely a privacy modifier.

As local variables are only able to be accessed from within the method, there is no need to add privacy modifiers to them.

Remove these modifiers and then try run your code again.

Incorrect Code Correct Code Explanation

Example

 
boolean productIsEven (int a, int b) {
	public int product = a*b; 
	return (product%2 == 0);
}
 
boolean productIsEven (int a, int b) {
	int product = a*b; // public modifier is removed
	return (product%2 == 0);
}

The public modifier is illegal for a local variable.

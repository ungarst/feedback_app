It looks like your code was unable to compile. Lets take a look why...

It appears that you have an out of place semicolon in your code. This is generally caused by one of two things:

  • Having an extra semicolon in your code. This most commonly occurs in at the end of a line where the programmer puts a semicolon before the last closing parenthesis (see example below).
  • Incorrect variable declaration where the programmer forgets to specify the variable type in front of the variable name.

Example of extra semicolon

Incorrect Code

int product = 8*7;
if ( product % 2 == 0;) { // Semicolon at the end of the comparison is not required
}

Incorrect Code

int product = 8*7;
if ( product % 2 == 0) { // Removing the semicolon allows the error to disappear.
	return true;
}

Example of incorrect declaration

Incorrect Code

hours; // Forgetting to declare the type of the variable at declaration

Correct Code

int hours; // Specifying the variable type at declaration fixes this error.

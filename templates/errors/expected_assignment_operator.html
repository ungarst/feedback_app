It looks like your code was unable to compile. Lets take a look why...

The Java compiler has suggested that you are missing an operator in your code. A lot of the time the missing operator could be an equals sign, but it can also be other operators like +, -, /, *, %, || and &&.

In cases where the operator should be an equals left part of the line looks like something that could be a variable name, and the right part looks like something that could be assigned to the left part so the compiler suggests inserting an equals sign as to assign the right part to the left.

In cases where the operator should not be an equals sign, have a go at putting one of +, -, /, *, %, || or && into the line to make the expression make sense..

Examples of potential assignment

Incorrect Code

// Most of these errors can be fixed by inserting an equals
productIsEven true;
string "hello";
string s;
word "" + firstLetter + words + lastLetter;

Correct Code

productIsEven = true;
string = "hello";
string = s;
word = "" + firstLetter + words + lastLetter;

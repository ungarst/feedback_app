It looks like your code was unable to compile. Lets take a look why...

It appears that you are misusing the new keyword. The new keyword is reserved for the construction of new objects.

If you are attempting to call one of your variables new then this is most likely where this error is coming from.

If you are not trying to call a variable new then you are attempting to construct an object incorrectly.

Object construction is done by calling new ObjectType (parameters) and the object must be assigned to a variable or used in a method call. If the object's constructor doesn't take any parameters then you still need to include empty parenthesis.

Example

Incorrect Code

int new = 10; // Cannot call a variable 'new'

new String first = itemsString.charAt(0); // We don't need to put 'new' before the object type on the left hand side of the assignment

String part = new String; // Need to include the parameters to the objects constructor.

Correct Code

int newVariable = 10; // Changed variable name from new

String part = new String (); // Include empty parenthesis as constructor takes no arguments.

String first = itemsString.charAt(0); // Remove 'new' before the object type on the left hand side of the assignment

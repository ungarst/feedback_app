It looks like your code was unable to compile. Lets take a look why...

It appears you have broken the Java naming conventions, meaning your code was unable to compile.

The full Java naming conventions can be found in the Java Tutorials but we've included a brief summary of them below which should help you solve your problem.

  • Variable names are case-sensitive.
  • Variable names can contain letters, numbers and underscores.
  • Variable names must begin with a letter.
  • Variable names must not be one of the Java keywords (e.g. int, new)

Example

Incorrect Code

int new = 10; // new is a keyword and so cannot be used as a variable name
String char = "c"; // char is also a keyword

double 2ndNumber = 4.9; // variable names cannot start with a number

Correct Code

int newVariable = 10;
String character = "c";

double secondNumber = 4.9;

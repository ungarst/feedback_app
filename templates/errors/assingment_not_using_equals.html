It looks like your code was unable to compile. Lets take a look why...

It appears that you are trying to assign to a variable incorrectly, and are somehow missing, or incorrectly using an equals sign.

Remember, variable assignment should look like variableName = value; (you will also need to put the variable type infront of the varible name if you haven't already declared it).

Examples

Incorrect Code

int x = 5;
int a 10; // Need to insert an equals sign so we are assigning 10 to a
boolean x == 10; // Need to give the boolean variable a name so we are assinging to something

Correct Code

int x = 5;
int a = 10;
boolean b = (x == 10);

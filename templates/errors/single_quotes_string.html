It appears you are using the wrong type of quotation marks for strings.

Strings are created using double quotation marks.

chars are one character only and are initialized using single quotation marks.

Example

Incorrect Code

String s = 'Programming is fun!'; // Single quotes are reserved for character constants

Incorrect Code

String s = "Programming is fun!"; // Double quotes are to be used for strings

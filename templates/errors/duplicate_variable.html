It seems like you are trying to create two variables with the same name in your method.

Remember, you only have to declare a variable once even if you are assigning to it in multiple places

Incorrect Code Correct Code Explanation

Example

 
int a = 5;
...
int a = 10;
 
int a = 5;
...
a = 10;

Redefining a to be an integer is not required. Once a has been defined for the first time, we can continue to assign to it without including the variable type.

This error may also be due to you redefining the arguments to the method that you are writing.

This is unnecessary as Java already knows these variables exist after seeing them in the method header.

Incorrect Code Correct Code Explanation

Example

 
public static int sum (int a, int b) {
	int a, b;
	return a + b;
}
 
public static int sum (int a, int b) {
	return a + b; 
}

a and b are declared inside the method, even though they are already declared in the method header. By removing the declaration, the code can now successfully compile.

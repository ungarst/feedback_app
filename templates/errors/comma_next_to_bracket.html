It appears your error is having a comma either directly after an opening brace or directly before a closing bracket.

Either this is a typo, or you are using it to try and skip some parameters in a method, which Java doesn't allow you to do. Try searching the method in google to see if there is a way to avoid entering that parameter.

Incorrect Code Correct Code Explanation

Example

 
int x = someMethod(,10); 
int x = someMethod(10);

Delete the comma to get rid of this error

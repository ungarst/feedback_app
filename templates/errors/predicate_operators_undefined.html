It looks like your code was unable to compile. Lets take a look why...

It appears that you are attempting to compare two different types.

Comparisons can only be made between things which are the same type e.g. comparing ints with ints or booleans with booleans.

Have look through your code at all instances of ==, !=, <=, >= and make sure that they are only been used to compare things of the same type.

Example

Incorrect Code

boolean b = false;
boolean c = false;
double d = 10.5;
if (b == d) { // Cannot compare b and d as they are not the same type
	return true;
}

Correct Code

boolean b = false;
boolean c = false;
double d = 10.5;
if (b == c) { // b and c are both boolean so they can be compared
	return true;
}

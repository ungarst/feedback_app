It appears that you have an incorrectly written if statement in your code.

Lets quickly review the format of an an if statement and see if it can help you fix your error.

if (condition) { 
		// Statements
	} else if (condition) { 
		// Statements
	} else { 
		// Statements
	}

The if condition and block are the only required parts.

It is also possible to have:

  • if statements
  • if, else statements
  • if, else if, else statements
  • if, else if statements

Ensure that all your if statements follow this format and then try compile your code again

Incorrect Code Correct Code Explanation

Example

 
int c = 4;
if (c > 6); { 
	return true;
} else { 
	return false;
}
 
int c = 4;
if (c > 6) {
	return true;
} else { 
	return false;
}

The semicolon after the if statement finishes the conditional area, and so all code after it will be executed regardless. The following opening curly brace opens a new block of code, which will always be executed. When it reaches the matching closing curly brace, the block of code is completed and execution will continue. The program then reaches the else condition, but it isn't preceeded by an if statement. As all else statementsneed to be preceeded by an if statement, Java throws an error.

It looks like your code was unable to compile. Lets take a look why...

It appears that you have an incorrectly written if statement in your code.

Lets quickly review the format of an an if statement and see if it can help you fix your error.

if (condition) { // The condition must evaluate to a boolean and must be wrapped inside parenthesis followed by a curly bracket
	// Statements
} else if (condition) { // The else if must be preceeded with a closing curly bracket, and then follow the same rules as the if statement
	// Statements
} else { // The else must be preceeded by a closing curly brace and followed by and opening curly brace
	// Statements
}

The if condition and block are the only required parts.

It is possible to have:

  • if statements
  • if, else statements
  • if, else if, else statements
  • if, else if statements

Ensure that all your if statements follow this format and then try compile your code again

Example

int x = 15;
int y;
if (x == 15) {
	y = 2;
} else if (x > 15) {
	y = 3;
} else {
	y = 1;
}

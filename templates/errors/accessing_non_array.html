It appears that you are attempting to use square brackets on a variable that is not an array.

Square braces are used for accessing elements, and so can only be used on variables of type array

int[] a = {1, 2, 3, 4, 5};
int second = a[1];

Many people encounter this error when trying to look at a single character from a String.

Although accessing strings using square brackets works in some languages, it doesn't in Java

If you want to access a single character from a string in Java, use the .charAt(int index) method.

Incorrect Code Correct Code Explanation

Example

 
String s = "I love using CodeWrite";
char secondCharacter = s[1];
 
String s = "I love using CodeWrite";
char secondCharacter = s.charAt(1);

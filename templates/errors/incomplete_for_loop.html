It appears that you have an incorrectly formatted for loop in your code.

The only requirements of a for loop is having two semicolons in parenthesis, but the correct structure of a for loop is:

for(initialization; boolean_expression; update)
{
   // statements
}

Where:

  • The initialization is declaring and setting the iterating variable to it's start value.
  • The boolean_expression is an expression that must return true for the statements in the loop to be executed.
  • The update is the change to the iterating variable that should happen after each time through the loop.

It is possible to rearrange this so the initialization and upadate are not within the parenthesis but this is the recommended way.

Example

Incorrect Code

int sum = 0;
for (int i = 0 ; i < 10 i++) { // Missing a semicolon after i < 10
	sum += i;
}

Correct Code

int sum = 0;
for (int i = 0 ; i < 10 ; i++) {
	sum += i;
}

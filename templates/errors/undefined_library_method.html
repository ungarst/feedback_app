It appears you are trying to call a library method that does not exist.

Library methods are methods that exist in java.lang, where the Math and String classes are found.

Ensure that the spelling of the method is correct. Java cannot tell if the method is spelt incorrectly, and will think it does not exist regardless.

If you are sure that the spelling is correct then the method may not exist. Try Google-ing the method to see you have the inputs right or the method actually exists.

	c = true;

	c = false;

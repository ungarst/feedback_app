It appears one of your if statements isn't written properly.

When we try to compare two things in an if statement, we need to use two equals signs.

Remember:

  • One equals sign is for assigning a value to a variable e.g. int x = 10
  • Two equals signs are for comparing two things e.g. x == 10
Incorrect Code Correct Code Explanation

Example

 
int x = 5;
int y = 13;
if ((x*y)%2 = 0) {
	return true;
} else {
	return false;
}
 
int x = 5;
int y = 13;
if ((x*y)%2 == 0) {
	return true;
} else {
	return false;
}

We are only using a single equals to compare the product of x and y modulo 2 with zero. If we are trying to compare then we need ot use two equals signs.

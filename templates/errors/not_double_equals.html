It looks like your code was unable to compile. Lets take a look why...

It appears one of your if statements isn't written properly.

When we try to compare two things in an if statement, we need to use two equals signs.

Remember:

  • One equals sign is for assigning a value to a variable e.g. int x = 10
  • Two equals signs are for comparing two things e.g. x == 10

Example

Incorrect Code

int x = 5;
int y = 13;
if ((x*y)%2 = 0) { // If we are trying to compare then we need ot use two equals signs
	return true;
} else {
	return false;
}

Correct Code

int x = 5;
int y = 13;
if ((x*y)%2 == 0) { // Correct way of comparing
	return true;
} else {
	return false;
}

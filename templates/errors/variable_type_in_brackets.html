It looks like your code was unable to compile. Lets take a look why...

It appears that when you were declaring your variables you wrapped the variable type in parenthesis, which you don't need to know.

Remove the parenthesis from around the variable type and try run your code again.

Example

Incorrect Code

(int) i = 50;
(String) s = "Hello";

Correct Code

int i = 50;
String s = "Hello";

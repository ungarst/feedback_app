It appears your code caused a casting error when we were compiling it for you. Casting errors occur when you try and cast an object to a type it can't be casted to (e.g. casting a string to an int).

Ensure that your variables are able to be casted from one type to another and try run your code again.

Incorrect Code Correct Code Explanation

Example

 
int x;
String s = "Hello, World!";
x = (int) s;
 
int x;
double d = 34.92;
x = (int) d;

Strings cannot be casted to an integer, so the incorrect code cannot compile. Doubles can be casted to integers (although some precision is lost) so the correct code can compile.

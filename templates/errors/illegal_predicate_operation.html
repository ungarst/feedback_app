It looks like your code was unable to compile. Lets take a look why...

It looks like you are using an illegal operator in your if statements.

Operations in if statements that are counted as illegal include

  • =! - The exclamation mark should be before the equals sign
  • => - The greater than sign should come before the equals sign
  • =< - The less than sign should come before the equals sign
  • << - This is the left bitwise shift, not much less than
  • >> - This is the right bitwise shift, not much greater than
  • \\ - Remember, if you are trying to have your if based on one condition OR an other, you need to use ||

Example

Incorrect Code

int x = 5;
if (x =< 10) { // Less than and equals signs are around the wrong way
	x = 10;
}

Correct Code

int x = 5;
if (x <= 10) { // Correct order of less than and equals signs
	x = 10;
}

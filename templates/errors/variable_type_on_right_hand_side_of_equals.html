It appears that when initializing one of your variables you are including variable type on the right hand side of the equals sign.

You don't need to do this as Java already knows if this is the right type of variable or not.

Remove the variable type from the right hand side of the equals sign and try run your code again.

Incorrect Code Correct Code Explanation

Example

 
int a = int 7; 
int a = 7;

We don't need variable type in front of the 7

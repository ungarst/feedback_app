It appears you are trying to call a method that doesn't exist. Let's see if we can fix this...

Remember we are only writing a single method, so if we are going to be calling methods then they will be a:

  • Static method from a library class (e.g. java.lang.Math)
  • A method on a variable that we have defined in the method we are writing

If we want to call a method on this variable, then we do it using variableName.methodName(parameters)

This error is also caused by misspelling Java keywords that are followed by parenthesis

Incorrect Code Correct Code Explanation

Example

 
int a = 5; 
int b = 12;
int c = max(a,b);
 
int a = 5; 
int b = 12;
int c = Math.max(a,b);

max is in the Math package so we need prefixed it with the package name

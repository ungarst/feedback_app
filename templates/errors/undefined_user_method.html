It looks like your code was unable to compile. Lets take a look why...

It appears you are trying to call a method that doesn't exist. Let's see if we can fix this...

Remember we are only writing a single method, so if we are going to be calling methods then they will be a:

  • Static method from a library class (e.g. java.lang.Math)
  • A method on a variable that we have defined in the method we are writing

If we want to call a method on this variable, then we do it using variableName.methodName(parameters)

This error is also caused by misspelling Java keywords that are followed by parenthesis

Example of calling a static method

Incorrect Code

int a = 5; 
int b = 12;
int c = max(a,b); // max is in the Math package so we need to prefix it with the package name

Correct Code

int a = 5; 
int b = 12;
int c = Math.max(a,b); // max is in the Math package so we need prefixed it with the package name

Example of calling a method on a variable

Incorrect Code

String s = "Java is awesome!!"
char c = charAt(s, 1); // Trying to get the character at index 1 of the string

Correct Code

String s = "Java is awesome!!";
char c = s.charAt(1); // charAt is a String method so we call it on the String s

Example of a misspelt Java keyword

Incorrect Code

int x = 10;
boolean b;
If (x > 5) { // if is spelt with a capital 'I'
	b = true;
} elseif (x == 5) { // There is no space between "else" and "if"
	b = true;
	x = 5;
} else {
	b = false;
}

Correct Code

int x = 10;
boolean b;
if (x > 5) { // The capital 'I' is now lower case
	b = true;
} else if (x == 5) { // There is now a space between "else" and "if"
	b = true;
	x = 5;
} else {
	b = false;
}

It looks like your code was unable to compile. Lets take a look why...

It appears you included the method header with the code you submitted.

When you submit your code, CodeWrite wraps what you write in a method header and puts it a class so that it can be tested.

Try removing the method header from the code you have written and submit again.

Example

Say you are writing the method boolean productIsEven (int a, int b) and you submit the following solution

boolean productIsEven (int a, int b) {
	return (a*b)%2 == 0;
}

CodeWrite will wrap your code within the given method header before it compiles it, so the compiled code is:

boolean productIsEven (int a, int b) {
	boolean productIsEven (int a, int b) {
		return (a*b)%2 == 0;		
	}
}

Simply remove the method header and corresponding curly brackets from your code and try again.

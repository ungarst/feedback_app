It looks like your code was unable to compile. Lets take a look why...

It seems like one of the lines in your code has an extra closing parenthesis.

Remember, every opening parenthesis must be matched with a closing parenthesis. This could be due to you accidentally including an extra closing parenthesis or forgetting to include an opening parenthesis.

Have a look at your code and ensure that all your brackets are matched, then try run your code again.

Example

Incorrect Code

String word = "Programming";

char firstLetter = word.charAt(0);
char secondLetter = word.charAt(word.length()-1);

String newWord = (secondLetter + word.substring(word.length())+firstLetter));
return newWord;

Correct Code

String word = "Programming";

char firstLetter = word.charAt(0);
char secondLetter = word.charAt(word.length()-1);

String newWord = (secondLetter + word.substring(word.length())+firstLetter);
return newWord;

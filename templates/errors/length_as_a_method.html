It appears you are trying to use .length() as a method on an array.

Make sure you remember that length is a field in an array so should be accessed using yourArrayName.length

The .length() method is applicable for Strings though, and so should be accessed using stringName.length()

Incorrect Code Correct Code Explanation

Example

 
int[] a = {1, 2, 3, 4, 5};
int len = a.length();
 
int[] a = {1, 2, 3, 4, 5};
int len = a.length;

Length is not a method and so should not be called with parenthesis after it.

It looks like your code was unable to compile. Lets take a look why...

It appears you are trying to use .length() as a method on an array.

Make sure you remember that length is a field in an array so should be accessed using yourArrayName.length

The .length() method is applicable for Strings though, and so should be accessed using stringName.length()

Example of finding the length of an array

Incorrect Code

int[] a = {1, 2, 3, 4, 5};
int len = a.length(); // Length is not a method and so should not be called with parenthesis after it

Correct Code

int[] a = {1, 2, 3, 4, 5};
int len = a.length;

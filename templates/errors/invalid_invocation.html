It looks like your code was unable to compile. Lets take a look why...

It appears that you tried to invoke a method on an object that the object didn't have.

Check the Java API to make sure that the method you are calling exists for that type of object and then try run your code again.

Example

Incorrect Code

char c = 'b';
char d = c.charAt(0); // The method charAt(int index) doesn't exist for type character

Correct Code

String c = "b";
char d = c.charAt(0); // The method charAt(int index) exists for type String
